# Домашнє завдання №5: Рекурсія, Замикання та Декоратори

Цей репозиторій містить розв'язання задач, для на засвоєння принципів рекурсії,
функціонального програмування (генератори, замикання) та використання декораторів у Python.

## Структура проєкту

* `task_1.py` — Оптимізація обчислення чисел Фібоначчі за допомогою замикання та кешування.
* `task_2.py` — Функція-генератор для аналізу рядка тексту та підрахунку загального прибутку.
* `task_3.py` — Скрипт для аналізу лог-файлів з можливістю фільтрації за рівнем логування.
* `task_4.py` — Апгрейд ДЗ 4 - Консольний бот-асистент із впровадженою обробкою помилок через декоратори.

---

## Опис завдань

### 1. Кешування чисел Фібоначчі
Реалізовано функцію `caching_fibonacci`, яка використовує механізм замикання для зберігання результатів попередніх обчислень

### 2. Генератор чисел у тексті
Створено функцію-генератор `generator_numbers`, яка за допомогою регулярних виразів знаходить усі дійсні числа у тексті.
Функція `sum_profit` використовує цей генератор для обчислення пыдсуику чисел.

### 3. Аналіз лог-файлів
Скрипт зчитує лог-файл та виводить статистику за рівнями (INFO, ERROR, DEBUG, WARNING). 
**Використання:**

python task_3.py /path/to/logfile.log [level]

*Підтримує необов'язковий аргумент для детального перегляду логів конкретного рівня.*


### 4. Бот-асистент з декораторами
Вдосконалено бота для керування контактами.
Додано декоратор `input_error`, який централізовано обробляє винятки:
* `ValueError` — при недостатній кількості аргументів.
* `KeyError` — якщо контакт не знайдено.
* `IndexError` — при некоректному зверненні до аргументів команди.
